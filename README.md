# Pixiv PBD Manager

[English README](README.en.md)

Pixiv PBD Manager 是一个本地 Pixiv 图片库管理工具。它可以扫描已有图库、记录艺术家、检查更新、下载新作品，也可以查找不同文件名、不同格式或不同分辨率下的相似图片。

这个 README 面向“只想使用软件”的用户。开发、命令行和实现细节放在 [docs/zh](docs/zh/) 里。

## 下载与安装

已提供 Windows x64 安装包：

1. 打开 [Releases](https://github.com/J-FPV/Pixiv-PBD-Manager/releases)。
2. 下载最新版本的 `Pixiv.PBD.Manager_*_x64-setup.exe`。
3. 运行安装包，然后从开始菜单启动 Pixiv PBD Manager。

一般用户推荐下载 `setup.exe`。`msi` 也会同时发布，主要适合需要 MSI 的安装场景。

安装版已经包含 Python 后端和依赖，不需要你另外安装 Python、Node.js 或 Rust。

## 它能做什么

- 扫描你的 Pixiv 图片目录，识别艺术家、作品 ID 和保存路径。
- 记录每个艺术家的 Pixiv ID、名字、已保存作品和可下载新作品。
- 检查单个或多个艺术家的更新，并显示每个艺术家有多少新图。
- 直接下载检查到的新作品，也可以打开 Pixiv 页面交给 Powerful Pixiv Downloader 下载。
- 识别没有艺术家 ID 的旧文件夹，并支持在线解析或手动指定 ID。
- 按排除目录跳过你不想扫描的文件夹。
- 收藏常看的艺术家，并用标签（分类）整理，支持拖拽批量打标签与按标签筛选。
- 图库视图：用缩略图网格浏览全部图片，可收藏、评分、标记整理状态，并按作者、文件夹、标签、评分等条件筛选。
- 查找相似图片，帮助发现跨平台下载造成的重复图。
- 支持缩略图预览和差异图对比。
- 推荐保留质量更好的版本，并将其余文件安全移入可恢复的隔离区。
- 支持中文 / English 界面切换。

## 推荐使用方式

1. 启动软件。
2. 在“设置”里添加你的图片下载目录。
3. 如有不想扫描的文件夹，加入排除目录。
4. 回到“艺术家”页，点击“扫描”。
5. 在扫描预览里确认要写入的变更。
6. 点击“检查更新”，查看哪些艺术家有新作品。
7. 选择需要的艺术家，点击“下载更新”；或者点击“打开选中”，用浏览器里的 Powerful Pixiv Downloader 下载。

## 第一次设置

打开“设置”页后，建议先检查这些项目：

- **下载目录**：你的 Pixiv 图片库目录，可以添加多个。
- **排除目录**：不想扫描的目录，例如杂图、非 Pixiv 图、浏览器缓存目录。
- **相似图片目录和排除目录**：相似图片扫描有独立的扫描目录和排除目录，会单独保存。
- **隔离目录**：清理重复图片时，文件先移动到这个图库外目录，不会直接永久删除。
- **数据库位置**：安装版默认保存在 `%APPDATA%\PixivPbdManager\.pixiv-pbd-manager\artists.json`，一般不用改。
- **浏览器路径**：如果默认浏览器不是安装了 Powerful Pixiv Downloader 的那个，可以在这里指定 Chrome / Edge。
- **浏览器用户数据目录**：不要选图片库目录。Chrome / Edge 会在用户数据目录里创建 `Default`、`ShaderCache`、`Safe Browsing`、`Webstore Downloads` 等文件夹。
- **Pixiv Cookie**：只有下载登录可见或 R-18 作品时才需要。Cookie 等同于登录凭证，使用前请阅读界面里的风险提示。

## 推荐 PBD 命名规则

如果你还在使用 Powerful Pixiv Downloader，建议让文件夹或文件名包含 Pixiv 艺术家 ID。推荐规则：

```text
pixiv/{user}-{user_id}/{id}-{title}
```

这样软件可以直接从文件夹名识别艺术家。旧目录如果只有作者名、没有 ID，也可以尝试开启在线解析或模糊搜索，但准确性会受文件名和 Pixiv 搜索结果影响。

## 下载更新

“下载更新”使用软件内置下载器，不是调用 PBD。它会下载“检查更新”后记录到数据库里的新作品，并保存到该艺术家的保存路径。

常见流程：

1. 先扫描本地图库，让软件知道每个艺术家保存在哪里。
2. 点击“检查更新”。
3. 表格“可下载”列出现数量后，点击“下载更新”。
4. 如果勾选了艺术家，只下载勾选艺术家的新作品；如果没有勾选，则下载所有有更新的艺术家。

设置里的“并行下载数”（1–5，默认 1）可以让多个作品同时下载；越高越快，但越容易触发 Pixiv 限流，建议保持 1–2。下载时任务窗口会为每个并行任务单独显示进度条。

如果你更想使用 PBD 的命名规则、过滤器或浏览器扩展流程，可以用“打开选中”打开艺术家页面，再在浏览器里使用 PBD。

## 收藏与标签

“艺术家”页支持收藏和标签（分类），方便整理大量艺术家。

- **收藏**：点击艺术家行最左侧的星标即可收藏 / 取消收藏，也可以右键菜单操作。点击工具栏的星标按钮可以只看收藏。星标列可排序。
- **标签栏**：工具栏和列表之间有一条可展开 / 收起的标签栏。点 `+` 新建标签，标签颜色会按名称自动分配。
- **打标签**：在列表里勾选一个或多个艺术家，然后把它们拖到某个标签上，即可批量打标签。也可以在“编辑”窗口里直接填写标签（逗号分隔）。
- **按标签筛选**：点击标签即可筛选，可同时选中多个标签（显示带有任一所选标签的艺术家）。
- **管理标签**：右键标签可以重命名或删除；删除会把该标签从所有艺术家上移除。

收藏与标签都会保存在 `artists.json` 里，不影响本地图片文件。

## 图库

“图库”页用缩略图网格浏览下载目录里的全部图片，把软件从“下载管理器”变成“素材管理器”。

- **建立目录**：首次打开点击「扫描图库」，软件会遍历下载目录、读取每张图的尺寸并解析 PID/作者，建成一个本地目录（`library_index.json`）。以后打开秒开；再次扫描只会重新读取大小或修改时间变化过的图。
- **筛选**：左侧可折叠筛选栏按作者、文件夹、标签、收藏、评分、整理状态、格式、方向（横/竖/方）、分辨率、年份筛选；顶部搜索框按文件名、路径、PID、作者和标签过滤。
- **收藏、评分与状态**：图片可收藏、设为 0–5 星，并标记为“参考价值高”“已用过”或“待整理”。这些信息只写入本地索引，不修改图片文件。
- **批量整理**：用缩略图左上角的选择按钮多选图片，然后统一添加/移除标签、修改收藏和评分、设置整理状态，或把已经抓取的 Pixiv 标签加入本地标签。
- **导出列表**：有选中图片时导出所选；没有选中时导出当前筛选结果。CSV 使用 UTF-8，可按作者、文件夹或标签筛选后直接导出。
- **详情**：点开任意缩略图查看大图与元数据，可编辑收藏、评分、状态和标签，或打开文件位置/Pixiv 来源页。
- **后台增量索引**：索引超过 6 小时、扫描目录变化或根目录出现新内容时，软件会在后台更新；已有图片会复用尺寸信息，不阻塞艺术家和相似图片操作。
- **图库体检**：点击工具栏的“图库体检”，检查数据库、失效保存路径、路径重叠、危险浏览器目录、隔离目录可写性和索引状态。体检只读，不会修改文件。

图库目录保存在 `library_index.json`，扫描时间和目录快照保存在旁边的 `library_index.meta.json`，都不影响本地图片文件。

## 相似图片

打开“相似图片”页，选择要扫描的目录，然后点击“查找相似图片”。软件会列出疑似重复组、显示缩略图，并按分辨率、文件大小、修改时间和路径推荐保留版本。

注意：

- 软件不会自动清理。确认后，所选文件只会移动到你指定的隔离目录。
- “可能相似”不会预选清理；每组始终至少保留一张图片。
- “隔离区”可以恢复文件，也可以手动永久删除。永久删除需要再次确认，且不会自动过期清理。
- 清理确认会分别列出本次保留和移入隔离区的文件；结果页按相似类型统计预计释放空间，隔离记录可直接打开对应任务目录。
- 可以忽略确认不是重复图的分组；组内文件发生变化后，该分组会重新出现。
- 不同分辨率、轻微压缩、不同格式的同图可能会被识别出来。
- 可以勾选“不比较同一 Pixiv 作品的拆分页”，避免 `{pid}_p0` 和 `{pid}_p1` 这类同一作品不同页互相干扰。
- 打开预览后可以在同组图片之间切换，并提供「单图 / 并排 / 滑块 / 差异」四种对比模式：并排会显示每张图的分辨率/大小/修改时间并标出建议保留的那张；支持滚轮缩放、拖动平移（并排时双图同步）；滑块可像「前后对比」一样擦出差异；差异图会先把两张图对齐到同一尺寸再做差分。支持方向键切换、Esc 关闭、数字键 1-4 切换模式。

## 常见问题

### 扫描不到作者怎么办？

优先检查你的目录或文件名里是否包含 Pixiv 艺术家 ID。最稳定的方式是使用上面的 PBD 命名规则。

如果旧目录只有作者名，可以在设置里开启在线解析；如果是手动命名的目录，可以尝试开启模糊搜索。扫描后仍未识别的文件夹会出现在“未识别文件夹”页，你可以手动指定艺术家 ID。

### 图片库里出现了奇怪的浏览器文件夹怎么办？

如果出现 `Default`、`Safe Browsing`、`ShaderCache`、`Webstore Downloads`、`Crashpad` 等文件夹，通常是浏览器用户数据目录被错误设置到了图片库里。

先关闭软件和浏览器，把浏览器用户数据目录改到图片库之外；确认这些文件夹不是你的图片目录后再手动清理。

### Cookie 必须填吗？

不必须。公开作品通常不需要 Cookie。只有登录可见、年龄限制或部分受限作品可能需要 Cookie。

如果要填写 Cookie，建议使用专门的小号，并在 Pixiv 网页端定期注销会话。

### 想从源码运行怎么办？

源码运行主要给开发者使用：

```powershell
pip install -e .
cd desktop
npm install
npm run tauri:dev
```

源码运行需要 Python、Node.js/npm、Rust/Cargo。日常使用建议直接安装 Release 里的安装包。
