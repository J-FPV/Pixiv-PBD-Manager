# 版本说明

[English](../en/release-notes.md)

## v0.1.9

这是一次以“素材管理”和大图库交互为重点的更新。安装版可以直接覆盖升级，`artists.json`、图库索引、相似图片结果、隔离记录和设置都会继续保留。

### 图库整理

- 图片支持收藏、0–5 星评分，以及“参考价值高 / 已用过 / 待整理”状态。
- 支持本地图片标签、将已有 Pixiv 标签复制为本地标签，以及多选后的批量编辑。
- CSV 导出会导出所选图片；没有选择时导出当前筛选结果。
- 新增图库体检，检查数据库、失效/重叠路径、危险浏览器目录、隔离目录和索引状态，不会修改文件。
- 索引过期或目录变化时可以后台增量更新，不阻塞艺术家和相似图片操作。

### 预览与相似图片

- 图库详情和相似图片预览会先完整显示长图、竖图，再支持滚轮缩放和拖动平移。
- 图库翻页按钮移到图片两侧，详情浏览更直接。
- 相似结果、隔离任务和任务内文件加入分页，降低大量记录同时渲染的开销。
- 隔离区继续保持“先移动、可恢复、永久删除需二次确认”的安全策略。

### 扫描、识别与任务控制

- 扫描可以取消，关闭扫描预览后可以从“扫描结果”再次打开。
- 扫描预览里的艺术家可以直接打开 Pixiv 页面，方便确认识别是否正确。
- 在线解析会从较新、中间和较旧位置抽取多个作品 ID，单个失效作品不会让整个文件夹识别失败。
- 改进本地文件处理、下载任务进度和资源管理器定位行为。

### 性能与稳定性

- 大图库的标签切换、筛选和侧栏计数改为延后重算，减少明显卡顿。
- 收藏、评分和整理状态在点击后立即反馈，后台按顺序写入，避免连续点击互相覆盖。
- 修复批量标签窗口中复选框错位的问题。
- stdin JSON 统一按 UTF-8 处理，并兼容 Windows PowerShell 5.1 写入的 UTF-8 BOM。

### 升级说明

- 推荐退出旧版本后再安装 `Pixiv.PBD.Manager_0.1.9_x64-setup.exe`。
- 用户数据默认仍位于 `%APPDATA%\PixivPbdManager\.pixiv-pbd-manager\`。
- 本版本不会主动删除图库文件；相似图片清理仍只会把确认的文件移入用户指定的图库外隔离目录。
